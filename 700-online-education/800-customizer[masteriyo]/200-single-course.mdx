# How to manage the Single Course using Masteriyo?

Using Online Education theme, you can manage the single course page with the help of Masteriyo plugin.

    Location: Appearance > Customize > Masteriyo > Single Course 


## Available Options

- **Title Position**: It allows you to choose a given options to manage the position of the title in the single course page using Masteriyo
    - Contents

    - Page Header

    ![Single Course-Title Position](https://user-images.githubusercontent.com/109593521/184597004-ae6e75f3-5e5e-46c4-97d1-1770db26c241.gif)

   

- **Sidebar Position**: It allows you to choose a given options to manage the position of the sidebar in the Single Course page using Masteriyo.
    - Right

    - Left

![Single Course-SIdebar Position](https://user-images.githubusercontent.com/109593521/184598918-23765769-7f83-42a5-a04d-25fb0e2bc0ad.gif)
