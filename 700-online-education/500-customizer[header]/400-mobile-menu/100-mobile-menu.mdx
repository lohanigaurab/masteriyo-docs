#  How to manage mobile menu?

Online Education theme allows you to manage the mobile by changing the color of meu text, background and close icon.

## Available options:

- **Menu**:
  - Color: It manages to add color of the menu text in mobile view.

- **Background**:
  - Color: It manages to add background color of the menu in mobile view.

- **Close Icon**
  - Color: It manages to add  color of the menu close icon in mobile view.
