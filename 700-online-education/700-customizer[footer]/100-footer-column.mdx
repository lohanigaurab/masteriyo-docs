# How to manage the Footer Column?

    Location: Appearance > Customize > Footer > Footer Column

Online Education theme als allows you to customize the Footer Column area using the options explained below:

## Background: It allows to manage **Background Color** or **Background image** as per your requirment for the Footer Column. follow the steps as mentioned below:

- From your **WordPress Dashboard**, go to the **Appearance > Customize > Footer > Footer Column** section.

- Choose the color for the background under **Background Color** option or add a image for the background under Background Image of the footer column.

- Click on **Publish**.

### Available Options

- **Background Color**: It manage to change the color of the background for the footer column.

- **Background Image**: It manage to add  the image of the background for the footer column.

- **Background Repeat**

- **Background Position**

- **Background Size**

- **Background Attachment**


## Colors: It allows to manage Colors of the text as per your requirment for the Footer Column.

Available Options

- Heading Color

- Text Color

- Link Color

- List Link Color


## Link Separator: It allows to manage link separator for the Footer Column.

- Enable

- Color
