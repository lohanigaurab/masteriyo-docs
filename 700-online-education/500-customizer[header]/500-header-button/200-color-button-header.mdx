# How to change the colors of the button of the header?

![Header Button - Colors](https://user-images.githubusercontent.com/109593521/183478245-0264a39a-2d6e-4985-81d9-ded2f7e77a42.png)


## Available Options:

-Colors: It manages to change the color and background of the button.

    - **Text**: Change the color of the text in the button.


    - **Background**: Set the background color of the button.
