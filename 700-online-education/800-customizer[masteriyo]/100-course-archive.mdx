# How to manage the Course Archive using Masteriyo?

Using Online Education theme, you can manage the course archive page with the help of Masteriyo plguin.

    Location: Appearance > Customize > Masteriyo > Course Archive

## Available Option

- **Title Position**: It allows you to choose a given options to manage the position of the title in the Course Archive page using Masteriyo
