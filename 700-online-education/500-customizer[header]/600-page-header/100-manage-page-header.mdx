#  How to manage Page Header?

This theme allows you to manage the Page Header of your site.

## Available Options

- **Title**
    - Enable: It allows you to enable the title in the page header.

    ![Enable Page Header](https://user-images.githubusercontent.com/109593521/184543568-d5a46206-88c7-422f-93ea-a1669b7622f5.png)

- **Breadcrumbs**
    - Enable: It allows you to enable the Breadcrumbs in the page header.
    
    ![Enable Breadcrumbs](https://user-images.githubusercontent.com/109593521/184543652-4003b58a-0555-462f-9615-08eb4b545036.png)       

- **Dimensions**:
    - Padding: It allows you to manage the padding i.e. Top, Right, Bottom, Left as per your requirement of the title and Breadcrumbsin the page header 
    
    ![Page Header Dimensions](https://user-images.githubusercontent.com/109593521/184543740-5d4388c8-7d58-4168-bba0-6fd6a05c23c7.png)

- **Colors**:
    - Background: It allows you to change the background color as per your requirement of the page header.
      
      ![Page Header Background](https://user-images.githubusercontent.com/109593521/184544121-df9c8202-6300-480c-82ba-502a7b2df196.gif)
  
    - Title: It allows you to change the title color as per your requirement of the page header.

      ![Page Header Title Color](https://user-images.githubusercontent.com/109593521/184544270-a7826f45-fa9e-42bc-b828-bd80485a590e.png)

    - Text: It allows you to change the text  color as per your requirement of the page header.

      ![Page Header Text Color](https://user-images.githubusercontent.com/109593521/184544361-e9fc694e-6e8a-4ad7-9057-cb85b2785192.png)

    - Separator: It allows you to change the Separator color as per your requirement of the page header.

      ![Page Header Separator Color](https://user-images.githubusercontent.com/109593521/184544427-5cf8a821-349e-4455-9e01-32d1651cd5e3.png)

    - Link: It allows you to change the link color as per your requirement of the page header.

      ![Page Header Link Color](https://user-images.githubusercontent.com/109593521/184544500-429da4d4-9a91-4e76-a8ad-bebe256337a5.png
