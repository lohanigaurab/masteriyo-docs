## How to manage the Sidebar Layout of the site? 

Online Education theme allows you to manage all the sidebar layour of the pages/posts in the site.  

### Available Options

- **Default**: It allows to choose options to manage the sidebar Layout in a default of the site.
    - Right Sidebar

    - No Sidebar 

- **Archive**: It allows to choose options to manage the sidebar layout in Archive page of the site.
    
    - Default:

    - Right Sidebar:

    - No Sidebar: 

- **Page**: It allows to choose options to manage the sidebar layout in Page of the site.
    
    - Default:

    - Right Sidebar:

    - No Sidebar: 


- **Single Post: It allows to choose options to manage the sidebar layout in single post of the site.
    
    - Default:

    - Right Sidebar:

    - No Sidebar: 

- **Masteriyo Courses Archive**: It allows to choose options to manage the sidebar layout in Masteriyo Courses Archive page of the site.
